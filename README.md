##설치 방법 / how to Install
1.php 언어 파일 8개를 내려 받으십시오. <br/>
2.php 언어 파일 8개를 qa_lang폴더의 ko폴더안에 넣어주십시오.<br/>
3.q2a의 어드민 항목에서 korean-한국어로 변경하십시오.
